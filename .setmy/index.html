URL syntax:
http://setmy.browsersize.com/<width>x<height>[!]

Examples:
http://setmy.browsersize.com/900x500
http://setmy.browsersize.com/750x550!
 Click on one of the links on the left to set your browser's size by going to a URL.

You can create your own links that specify arbitrary dimensions. Start with "http://setmy.browsersize.com/", append the width, an "x" as the delimiter and finally the height. A "!" at the end makes the browser go back to the page that you were previously on.
To set your browser's size to a custom size, enter the width and height and click Set.
